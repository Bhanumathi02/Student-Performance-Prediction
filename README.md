# 🎓 Student Performance Prediction

A Machine Learning web application that predicts student performance using Python, Flask, and Scikit-learn.

## 🚀 Features
- Predict student performance
- User-friendly interface
- Real-time prediction
- Responsive design

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
